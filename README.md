> ## 概要 ##
 1. 这是一个php注释生成接口文档调试工具
 2. 支持模块化
 3. 生成文档支持权限验证
 4. 可指定模块生成链接


> ## 安装 ##
* **github克隆到项目根目录下**
```git
git clone https://github.com/php-xuan/apidoc.git
```
* **或者composer安装**
```composer
composer require guozhaoxuan/php-apidoc
```
